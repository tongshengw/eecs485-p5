System for hosting service providers
Hosting Controller
Developer(s) Hosting Controller, Inc.
Initial release 1999
Stable release
10.36 / March 2024
Operating system Windows , Linux
Type Cloud Automation
License Proprietary
Website http://www.HostingController.com

Hosting Controller is a multi-purpose cloud control panel and Active Directory synchronization solution that operates out of Canada and offers SaaS , Active Directory , Virtual Machine automation and Migration services to enterprises and service providers worldwide.

Overview

[ edit ]

Hosting Controller Cloud Automation Solution is a business and operational support system for hosting service providers , including shared web hosting for both Linux and Windows . It also supports SaaS and Enterprise hosting services like Microsoft Exchange Server , Virtualization , SharePoint , Active Directory Synchronization and Email Migration .

Latest Release

[ edit ]

The current version ofHosting Controller 10came out in February 2017. The most latest release of Hosting Controller 10.36 came outin March 2024 and offers a hybrid cloud control panel that lets service providers and enterprises manage their private clouds. The latest release also adds support for some of the most popular public clouds such as AWS and Microsoft Azure. [ 1 ]

HC Enterprise Modules

[ edit ]

The modules currently available for live production are:

- Microsoft Exchange 2010 [ 2 ]
- Microsoft Exchange 2013
- Microsoft Exchange 2016
- Microsoft Exchange 2019
- Hybrid Cloud Control Panel
- Skype for Business Module
- SharePoint Module
- Website Hosting Module
- VPS/VM Automation Module ( Hyper - V , VMware ) [ 3 ]
- RDS Module

Add-ons

[ edit ]

Hosting Controller supplements its control panel with a number of Add-ons. The supported Add-ons are:

- RebuildXpress (RBX)
- One Click Install Applications

Other Products

[ edit ]

Other Hosting Controller Inc. products include:

  • Efflux Migrator [ 4 ]
  • Mail Data Migrator [ 5 ]
  • ExSign Email Signature Software [ 6 ]
  • AD Connect Sync [ 7 ]
  • AD Self Service Portal [ 8 ]

Support for other applications

[ edit ]

Hosting Controller supports a number of other applications. Supported applications include:

- WHMCS
- PayPal
- SpamExperts
- LiteSpeed Web Server
- CloudLinux
